# Sales Trends Analysis

This repository contains a data analysis project focused on identifying business sales trends using historical sales data. The goal of this project is to explore patterns, understand key factors influencing sales, and extract insights that can support decision-making for businesses.

---

## Project Overview

The analysis focuses on understanding how sales behave over time, across categories, locations, and customer segments. The notebook includes data cleaning, exploratory analysis, visualizations, and interpretation of findings in a structured manner.

---

## Repository Structure

* **Business_sales_EDA.csv**
  Contains raw sales data used for exploration and analysis.

* **Untitled5.ipynb**
  Jupyter Notebook with complete code, visualizations, and step-by-step analysis.

---

## Key Areas Covered

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Trend analysis across months, regions, and product types
* Identification of high-performing and low-performing segments
* Visualization of sales patterns using charts and graphs
* Summary of insights and potential recommendations

---

## How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/your-username/sales-trends-analysis
   ```

2. Install required libraries:

   ```
   pip install pandas numpy matplotlib seaborn
   ```

3. Open the Jupyter Notebook:

   ```
   jupyter notebook Untitled5.ipynb
   ```
